# 前端跟踪埋点模块

> 通过应用主动埋点，记录应用程序运行过程中业务模块父子调用链及依赖关系，根据条件触发记录上传。

## 安装

```bash
tnpm install filofax
```

## 使用

```js
var Filofax = require('filofax');

// create global instance
var trace = new Filofax();

// 根节点埋点
trace.rootShot();

// 子节点埋点
trace.shot();

// 错误埋点，dump error 相关函数调用信息
var err = new Error('something wrong');
trace.dump(err);
```
